# 🩺 AI-Powered Personal Health Dashboard

Modern AI healthcare dashboard with Diabetes Prediction, Explainable AI (XAI), Health Analytics, Admin Dashboard & PDF Reports.

---

# 🚀 Features

## 🧠 Diabetes Prediction System
- Interactive health parameter sliders
- Real-time AI prediction
- Risk gauge visualization
- Confidence score display

## 🔍 Explainable AI (XAI)
- SHAP-style feature importance
- Transparent model reasoning
- Impact-based analysis

## 📊 Health Analytics
- Glucose trend charts
- BMI tracking
- Blood pressure monitoring
- Activity analytics

## 🏥 Admin Dashboard
- Patient statistics
- Prediction history
- Model performance metrics
- Risk distribution charts

## 📄 PDF Medical Reports
- Print-ready export
- Medical record formatting

---

# 🎨 Design Highlights

- Dark medical-tech UI
- Glassmorphism dashboard cards
- Animated gradient backgrounds
- Responsive mobile design
- Interactive donut charts
- Smooth micro animations
